Entrance Animations

Fade Animations

Six entrance variants powered by GSAP + ScrollTrigger. Customize duration, delay, blur, and distance with data-attributes.

fade-in

Default upward fade with 16px translate.

data-animate="fade-in"

fade-in + preset

Use a preset to apply a curated bundle of duration, blur, and distance settings.

data-animate="fade-in" data-animate-preset="dramatic"

fade-in-left

Slides in from the left with opacity.

data-animate="fade-in-left"

fade-in-right

Slides in from the right with opacity.

data-animate="fade-in-right"

fade-in-down

Drops in from above with opacity.

data-animate="fade-in-down"

fade-in-scale

Scales up from 95% with opacity.

data-animate="fade-in-scale"

Webflow How to use
1Select any element in the Designer
2Open Settings (gear icon) → Custom Attributes
3Add the attribute shown in the DOM tree
4Optional: add data-animate-preset with subtle, dramatic, slow, or fast
<!-- Any element --> <div data-animate="fade-in"> Your content </div>
<!-- With options --> <div data-animate="fade-in" data-animate-duration="0.8" data-animate-delay="0.2" data-animate-blur="4">
Variants: fade-in, fade-in-left, fade-in-right, fade-in-down, fade-in-scale. All values are pure numbers — no px or s units needed.

Reveal Animations

Clip-path based reveals that unmask content from a chosen direction. Great for images and bold visual blocks.

reveal-up

data-animate="reveal-up"
reveal-down

data-animate="reveal-down"
reveal-left

data-animate="reveal-left"
reveal-right

data-animate="reveal-right"
Webflow How to use
1Best on images, hero blocks, and full-width sections
2Add data-animate with the reveal direction
3Optionally customize data-animate-duration (default: 0.8)
<img data-animate="reveal-up" src="hero.jpg" />
<!-- Directions --> reveal-up | reveal-down reveal-left | reveal-right
Tip: Uses clip-path to "unmask" the element from the chosen direction. Great for dramatic image entrances.

data-animate-stagger="0.1"   data-animate-blur="4"
Webflow How to use
1Create a parent container (Grid, Flex, or Div Block)
2Add data-animate-stagger on the parent only
3Children animate in sequence automatically — no attributes needed on them
<div data-animate-stagger="0.1"> <div> Card 1 </div> <!-- delay: 0s --> <div> Card 2 </div> <!-- delay: 0.1s --> <div> Card 3 </div> <!-- delay: 0.2s --> <div> Card 4 </div> <!-- delay: 0.3s --> </div>
Tip: CMS Collection Lists work perfectly as stagger parents. Add optional data-animate-blur on the parent for a soft blur entrance.

Animated number counters with prefix, suffix, decimal, and separator support. Triggered on scroll into view.

Animation Types

data-animate="count-up"

data-animate-prefix="$" data-animate-decimals="1"
Webflow How to use
1Create a Text Block and type the target number
2Add data-animate = count-up
3Add prefix, suffix, or decimals as needed
<!-- Simple --> <span data-animate="count-up">500</span>
<!-- With formatting --> <span data-animate="count-up" data-animate-prefix="$" data-animate-suffix="M" data-animate-decimals="1"> 1.2 </span>
Tip: The counter reads the number from the text content. Just type it naturally — commas, prefix, and suffix are handled by attributes.

Parallax

Depth-creating motion tied to scroll position. Vary speed and direction per element for layered compositions.

data-animate="parallax" data-animate-speed="0.2" data-animate-direction="up"
Webflow How to use
1Select a decorative element inside a section
2Add data-animate = parallax
3Set data-animate-speed (0.1 subtle → 0.6 dramatic)
<section> <img data-animate="parallax" data-animate-speed="0.3" data-animate-direction="up" /> <img data-animate="parallax" data-animate-speed="0.1" data-animate-direction="down" /> </section>
Tip: Stack multiple elements with different speeds to create depth. Automatically reduced on mobile.
GSAP Effect

Horizontal Scroll

Pin a section and scroll through panels horizontally. Each panel is a full viewport width.

data-animate="horizontal-scroll"
Webflow How to use
1Create a Section with the attribute
2Inside: one Div Block set to Flex, no wrap
3Inside the wrapper: panels at 100vw × 100vh
<section data-animate="horizontal-scroll"> <div style="display:flex"> <div class="panel"> Panel 1 </div> <div class="panel"> Panel 2 </div> <div class="panel"> Panel 3 </div> </div> </section>
Tip: Set overflow: hidden on the section. Falls back to native horizontal scroll on mobile (<768px).

Panel 1

The section is pinned and this wrapper scrolls horizontally as you continue scrolling down.

Panel 2

Each panel occupies the full viewport width for an immersive experience.

Panel 3

Combine with fade-in or stagger on panel contents for layered choreography.

Panel 4

On mobile, the section falls back to native horizontal overflow scrolling.

data-animate="pin" data-animate-distance="400"

This section is pinned

It stays fixed in the viewport for 400px of additional scroll distance before releasing.

Webflow How to use Pin
1Select a Section you want to pin
2Add data-animate = pin
3Set scroll distance with data-animate-distance
<section data-animate="pin" data-animate-distance="400"> <!-- Stays pinned for 400px of scroll --> <div> Your pinned content </div> </section>
Tip: Great for storytelling — the section "sticks" while the user scrolls through context.

Zoom Pin

Image starts at reduced scale and zooms to fill the viewport while the section is pinned. Scroll-driven with smooth scrub.

data-animate="zoom-pin"

Zoom Pin

This content scales from 60% to fullscreen as you scroll

Webflow How to use Zoom Pin
1Create a Section with data-animate="zoom-pin"
2Inside: one child element (image, div, or video) that will zoom
3Set scroll distance with data-animate-distance
<section data-animate="zoom-pin" data-animate-distance="600"> <img src="hero.jpg" /> </section>
Tip: Use data-animate-scale="0.4" for a more dramatic zoom or data-animate-radius="40" for rounder corners at start.

Webflow How to use Cards Stack
1Create a parent container with data-animate="cards-stack"
2Add child cards with explicit height set via class

Webflow How to use
1Create a wrapper Div Block with the attribute
2Inside: one track Div Block set to Flex, no wrap
3Add your items inside the track — they auto-duplicate for the loop

Interactive Effects

Magnetic Button Hover Me Get Started Learn More

Move your mouse over the buttons

data-animate="magnetic"   data-cursor="hover"
Webflow How to use
1Magnetic: add data-animate="magnetic" to any button or link
2Cursor: create a fixed Div Block at the end of the body
3Hover effect: add data-cursor="hover" on interactive elements
<!-- Magnetic button --> <a data-animate="magnetic" data-cursor="hover"> Get Started </a>
<!-- Custom cursor (end of body) --> <div data-animate="cursor" data-animate-size="20"></div>
Tip: Both effects are auto-disabled on touch devices. Use data-cursor="hide" to hide the cursor over specific areas.

Text Split

Split text into characters, words, or lines and animate each piece individually. Powered by GSAP SplitText plugin.

Split by Characters

Each character animates individually with a stagger delay.

data-animate-type="chars"

Split by Words

Each word fades in with a stagger for a reading-friendly reveal.

data-animate-type="words"

Split by Lines for a Cinematic Reveal Effect

Entire lines animate in sequence — great for paragraphs and hero text.

data-animate-type="lines"
data-animate="text-split" data-animate-type="chars" data-animate-stagger="0.03"
Webflow How to use
1Select a Heading or Text Block
2Add data-animate = text-split
3Set the split type and stagger delay
<h1 data-animate="text-split" data-animate-type="chars" data-animate-stagger="0.03"> Your Headline </h1>
<!-- Types: chars | words | lines --> <!-- Add data-animate-mask="true" --> <!-- for clipping reveal effect -->
Tip: SplitText is included in Webflow's native GSAP. Use chars for hero headings, lines for paragraph reveals.

Webflow How to use
1Create filter buttons with data-filter
2Create a Grid container with data-animate="flip-container"
3Each item needs data-flip-id + data-category
<!-- Filter buttons --> <button data-filter="all"> All </button> <button data-filter="design"> Design </button>
